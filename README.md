# 💰 ChurnSense Pro – AI Retention Intelligence System

## 🚀 Overview
ChurnSense Pro is an AI-powered decision system that predicts customer churn and optimizes retention strategies using profit-based analytics.

## 🔥 Features
- Churn Prediction using ML (Ensemble Models)
- Profit Optimization Engine
- Customer Segmentation (KMeans)
- Actionable Retention Dashboard
- Live Customer Risk Prediction

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas, NumPy

## 📊 Business Impact
- Reduces revenue loss
- Optimizes marketing spend
- Enables data-driven decisions
